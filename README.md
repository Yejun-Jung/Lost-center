# 1️⃣ 프로젝트 개요 ⭐
- 경복대학교 학생들을 위한 분실물 및 습득물 정보 공유 웹사이트다.
- 미디어 쿼리를 활용한 반응형 웹 구현 능력을 기르는 데 중점을 두었다.

----

# 2️⃣ 화면 구성
- 상단에 Header영역, 중간에 Search, Found/Lost Section을 포함한 본문영역, 하단에 Footer영역으로 나누어 구성했다.

----

# 3️⃣ HTML 작성 시 신경 쓴 점
- header, main, section, footer 등을 사용하여 콘텐츠의 역할에 맞게 구조를 설계했다.

----

# 4️⃣ CSS 작성 시 신경 쓴 점
- Flexbox를 활용해 유연한 레이아웃을 구성하고, 카드 UI를 통해 정보의 가독성과 시각적 일관성을 높였다.
- gap, padding 등 세밀한 간격 조정과 미디어 쿼리를 통해 최적화된 정렬을 구현했다.

----

# 5️⃣ 구현 중 문제와 해결 ⭐
- 고정 너비(px)를 사용한 카드가 모바일 환경에서 부모 요소보다 커지거나, 오른쪽 여백이 남아버리는 현상이 발생했다.
- calc() 함수를 사용하여 카드 너비를 유동적으로 계산하고, wrap을 적용하여 화면 크기에 따라 카드가 자연스럽게 줄바꿈되도록 처리했다.

----

# 6️⃣ 실습을 통해 확인한 점
- 미디어 쿼리를 사용할 때 단순히 크기만 줄이는 것이 아니라, 사용자 환경에 맞춰 메뉴 구조나 카드 배열을 완전히 재구성하는 과정의 중요성을 이해했다.
- 배경 컬러와 폰트 스타일을 공통 스타일시트(common.css)로 관리함으로써, 여러 페이지 간의 시각적 통일성을 유지하여 작업의 효율성을 높이는 방법을 익혔다.

----

# 7️⃣ 개선할 점
- 각 페이지마다 동일한 header와 footer가 반복되고 있는데, 이를 하나로 관리하여 수정 시 모든 페이지에 한 번에 적용되도록 구조를 개선하고 싶다.
- 검색창 자동 완성 기능이나 카테고리별 필터링 버튼 등, 정보를 더 빠르게 찾을 수 있도록 돕는 UI 요소들을 추가로 설계해보고 싶다.
